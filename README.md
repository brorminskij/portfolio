# My personal portfolio v1

## Tech Stack

**Client:** Nextjs, Typescript & Chakra UI

**Server:** Node & Express

## Run Locally

Clone the project

```bash
  git clone https://github.com/brorminskij/portfolio.git
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
npm install
# or
yarn install
```

Start the server

```bash
npm run dev
# or
yarn dev
```
